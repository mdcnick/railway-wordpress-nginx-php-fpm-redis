# Repo analysis skill

Use this when learning or re-learning the repository structure.

## Process

1. Start with root files.
2. Check package manager files.
3. Identify the framework, runtime, and deployment model.
4. Map folders at a useful level.
5. Do not read every file.
6. Update `_config/repo-map.md`.
7. Update `_config/learned-context.md` with durable facts.

## Output

Write a short repo learning summary to the current stage output folder.
